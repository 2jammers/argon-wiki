Learn about Argon and why you should use it!

## What is Argon?

It's a feature-full tool that allows use of external editors, version control systems, automated workflows and help manage both small and huge projects. It's made for professional users as well as for beginners!

## Why use Argon?

Because it takes very little time and effort to start using but brings ton of advantages:

### External Editors

Dive into the world of fast coding with many helpful extensions and exciting themes with these recommended editors:

- [Visual Studio Code](https://code.visualstudio.com/) - extremely customizable and extendable
- [Sublime Text](https://www.sublimetext.com/) - ultra fast and simple
- [Zed](https://zed.dev/) - brand new and smart (macOS only)
- [Neovim](https://neovim.io/) - terminal based and scriptable

### Tooling

Make your code better and prettier with this awesome community created tools:

- [Luau LSP](https://github.com/JohnnyMorganz/StyLua) - write code faster with smart competition
- [StyLua](https://github.com/JohnnyMorganz/StyLua) - write consistent and beautiful code with ease
- [selene](https://kampfkarren.github.io/selene/) - statically analyze your code to find potential problems
- [Wally](https://wally.run/) - easy and intuitive package manager for Roblox
- [roblox-ts](https://roblox-ts.com/) - seamless TypeScript to Luau compiler

### Version Control

Never lose your code again and keep an eye on changes made to your project by using [Git](https://git-scm.com/) with one of these hosts:

- [GitHub](https://github.com/) - most popular and flexible, the best overall
- [GitLab](https://about.gitlab.com/) - AI powered, focused on security and deployment
- [Gitea](https://about.gitea.com/) - self hosted and private

## Alternatives

There are other good Argon-like tools out here:

- [Rojo](https://rojo.space/) - most popular and advanced, similar to Argon
- [rbxmk](https://github.com/anaminus/rbxmk) - focused on file processing
- [Lync](https://iron-stag-games.github.io/lync/) - focused on automation
